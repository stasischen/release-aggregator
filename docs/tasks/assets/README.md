# Assets

這裡放可重用的 reference 類文件，不視為 active task 本體。

## Common Types

- architecture / contract / spec / proposal
- schema / index / mapping / inventory
- guide / brief / note / report
- mockup / prompt / seed / profile

## Rule

- 這裡的檔案通常只會被引用，不會作為任務進度主檔
- 若某份文件開始追蹤執行進度，應移回 `docs/tasks/` 根層 active working set
