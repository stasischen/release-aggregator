# KO A1 Anchor-First Seed (L01-L05)

Scope: absolute beginners, anchor-first only.  
This package intentionally excludes practice nodes (`P1~P6`) for now.

## L01 第一次見面：打招呼與自我介紹
- Anchor: 新生報到櫃台（dialogue）
- Core can-do:
  - 用 `안녕하세요` 打招呼
  - 用 `저는 ...예요` 自我介紹
  - 聽懂 `이름이 뭐예요?`
- Grammar notes:
  - `N + 이에요/예요`
  - 基本問候語（안녕하세요/반가워요）
  - 名字問句（이름이 뭐예요?）

## L02 你從哪裡來：國籍與出身地
- Anchor: 新生聊天（dialogue）
- Core can-do:
  - 問來源地
  - 說國籍
- Grammar notes:
  - `N에서 왔어요`
  - `N 사람이에요`
  - `어디에서 ...?`

## L03 校園問路：地點與存在
- Anchor: 找教室（dialogue）
- Core can-do:
  - 問地點
  - 聽懂位置回答
- Grammar notes:
  - `{place}이/가 어디예요?`
  - `있어요`（在/有）
  - 位置副詞（여기/저쪽）

## L04 便利商店：詢問有沒有與點單
- Anchor: 便利商店購物（dialogue）
- Core can-do:
  - 問有無（있어요?）
  - 禮貌請求（주세요）
- Grammar notes:
  - `{item} 있어요?`
  - `{item} 주세요`
  - 數量單位入門（한 병 / 한 잔）

## L05 時間與約定：今天幾點有課
- Anchor: 課前約見面（dialogue）
- Core can-do:
  - 問時間
  - 說明行程
- Grammar notes:
  - `지금 몇 시예요?`
  - `{time}에 {plan}이/가 있어요`
  - `...에서 만나요`
