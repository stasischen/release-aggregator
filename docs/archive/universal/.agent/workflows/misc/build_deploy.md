# Skill: Build & Deployment (構建與部署)

用於執行 Flutter 應用程式的構建、打包與部署。
Executes build, packaging, and deployment for the Flutter application.

## 🧭 Direction (方向)

1. **環境檢查 (Env Check)**: 確認 Flutter/CMake 環境。 (Verify Flutter/CMake environment.)
2. **依賴安裝 (Install Deps)**: `flutter pub get`。
3. **執行構建 (Build Execution)**:
   - **Windows**: `flutter build windows` (若失敗請參閱資源 / See resources if failed).
   - **Android/iOS**: TBD.
