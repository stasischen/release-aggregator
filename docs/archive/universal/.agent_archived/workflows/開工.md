# /開工 (Session Start)

**Start Here: Choose Your Path**
請選擇適合您當前工作的流程。
Please select the workflow that matches your current task.

由于專案規模更動，開工流程已分拆為 App 開發與內容生產兩條獨立路線，以減少上下文干擾。

Due to project scale, the session start worksflow has been split into separate App and Content tracks to reduce context switching.

## 📱 A. App 開發 (App Development)
**Use Command**: `/開工_app`

適用於：
- Flutter UI/UX 開發
- 功能實作 (Feature Implementation)
- App Bug Fixes
- 平台特定問題 (iOS/Android/MacOS)

## 📚 B. 內容生產 (Content Production)
**Use Command**: `/開工_content`

適用於：
- 課程編寫 (Course Creation)
- 字典維護 (Dictionary Maintenance)
- 內容審計 (Content Audit)
- 數據映射與驗證 (Mapping & Validation)

---

## ❓ 不確定？ (Unsure?)

如果不確定該選哪一個，請先執行 `/開工_content` 進行環境檢查，因為內容數據通常是 App 的上游依賴。
If unsure, run `/開工_content` first, as content data is often an upstream dependency for the App.
