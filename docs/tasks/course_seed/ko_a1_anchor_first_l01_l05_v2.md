# KO A1 Anchor-First Seed (L01-L05) — v2

Policy:
- 每課 2 anchors：`1 dialogue + 1 short text`
- 兩個 anchors 必須同主題
- 每課 3 個 grammar/pattern notes
- 先不做練習節點（P 節點後續再接）

## L01 新生報到：打招呼與自我介紹
- Dialogue: 國際處報到櫃台（8句）
- Text: 新生報到須知（3句）
- Grammar focus:
  - 안녕하세요（禮貌開場）
  - 저는 ...이에요/예요（自我介紹）
  - 이름이 뭐예요?（姓名詢問）

## L02 在學校裡問路：找教室與圖書館
- Dialogue: 新生找教室（8句）
- Text: 校園地圖提示（3句）
- Grammar focus:
  - {place}이/가 어디예요?
  - {location}에 있어요
  - 옆 / 맞은편（位置詞）

## L03 咖啡店點單：有沒有與請求
- Dialogue: 校園咖啡店點單（9句）
- Text: 咖啡店迷你菜單（3句）
- Grammar focus:
  - {item} 있어요?
  - {item} 주세요
  - 얼마예요?

## L04 時間與行程：今天幾點有課
- Dialogue: 課前約見面（8句）
- Text: 今日課表（3句）
- Grammar focus:
  - 지금 몇 시예요?
  - {time}에 {plan}이/가 있어요
  - ...에 만나요

## L05 聽不懂時怎麼辦：請再說一次
- Dialogue: 社團報名櫃台（8句）
- Text: 社團報名提醒（3句）
- Grammar focus:
  - 천천히 말해 주세요
  - 다시 ... 주세요
  - 알겠습니다 / 감사합니다（修復後收尾）
