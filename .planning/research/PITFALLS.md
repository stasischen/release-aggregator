# Research: Pitfalls

## Pitfall 1
Protocol is documented but not enforced.
- Prevention: add check commands and verification artifacts.

## Pitfall 2
State duplication causes divergent truth.
- Prevention: define `.planning/` as operational source and root files as summary layer.

## Pitfall 3
Cross-repo work bleeds into one session.
- Prevention: force one-phase-one-repo boundaries and phase-specific handoff notes.
