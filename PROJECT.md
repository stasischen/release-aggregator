# PROJECT

## Vision
Build and operate the Lingo multi-repo system as a reliable control tower:
- predictable cross-repo delivery
- clear ownership boundaries
- reversible and auditable changes

## Mission
Coordinate `content-ko`, `content-pipeline`, `core-schema`, `lingo-frontend-web`, and `lllo`
through a shared protocol that reduces context loss and execution drift.

## Principles
- Spec first, then execution.
- One phase, one repo.
- Atomic commits with evidence.
- State is written to files, not chat memory.
