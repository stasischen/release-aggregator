## 4. Content Components (Detailed Examples)

## 4.1 Immersion Input A: Ordering Dialogue

- `candidate_type`: `lesson`
- `content_form`: `dialogue`
- `learning_role`: `immersion_input`
- `skill_focus`: `integrated`
- `output_mode`: `none`

### Scenario

學習者在咖啡廳櫃台點飲料與甜點，店員會問內用外帶、冰熱、付款方式。

### Korean Sample Dialogue (Short)

店員: 어서 오세요. 주문하시겠어요?  
顧客: 아이스 아메리카노 한 잔이랑 치즈케이크 하나 주세요.  
店員: 매장에서 드세요, 아니면 포장이세요?  
顧客: 포장이요.  
店員: 결제는 카드로 하세요?  
顧客: 네, 카드로 할게요.

### Reviewer-facing zh-TW Summary

- 一段標準點餐對話，涵蓋點品項、數量、內用外帶、付款方式四個核心回合。

### Can-do Links

- 點餐指定品項與數量
- 回應店員提問（內用/外帶）
- 確認付款方式

### QA Risks (draft)

- A1 風險：如果一次放太多菜單詞彙，會壓縮回合理解
- 控制：v1 限制 2~3 個品項、1 個付款方式

- `candidate_type`: `lesson`

## 4.5 Grammar Note A: Sentence Frames (Pattern-first)

- `candidate_type`: `grammar_note`
- `learning_role`: `structure_pattern`
- `skill_focus`: `writing`

### Goal

提供點餐必需的句框與替換槽位，讓學習者先能「拼句」。

### Core Frames (zh-TW + slots)

1. 點餐請求
- `___ 한/두 ___ 주세요.`  
- 用途：點品項 + 數量

2. 加法連接（初級簡化）
- `___이랑/하고 ___ 주세요.`

3. 偏好/限制（初級模板）
- `___ 말고 ___ 주세요.`（先作模板理解）

4. 付款方式
- `카드로 할게요.` / `현금으로 할게요.`

### Slot Sets (for controlled output)

- 品項：아메리카노 / 라떼 / 케이크 / 빵
- 數量：한 잔 / 두 잔 / 하나
- 選項：아이스 / 핫 / 포장 / 매장
- 付款：카드 / 현금

## 4.6 Grammar Note B: Minimal Grammar (Output-serving)

- `candidate_type`: `grammar_note`
- `learning_role`: `structure_grammar`
- `skill_focus`: `reading`

### Include (only)

1. 請求句常用模板（固定搭配）
- 初級先當功能句，不延伸完整文法分類

2. 否定/排除（偏好約束）
- 例如「不要冰」「不要某品項」的模板化用法

3. 數量 + 單位基本搭配
- 一杯/兩杯/一個（避免全量數詞系統講解）

4. 確認問句尾句（功能導向）
- 用於回應店員確認

### Exclude (for this unit)

- 詳細敬語層級比較
- 不相干時態或複雜連接語

## 4.7 Dictionary Pack: Functional Phrases & Collocations

- `candidate_type`: `dictionary_pack`
- `skill_focus`: `integrated`

### Pack Goal

不是做「單字表」，而是提供能直接放進任務輸出的詞塊。

### Sections

1. Menu Items（菜單品項）
- 아메리카노
- 라떼
- 케이크
- 빵

2. Quantity / Classifier Chunks（數量詞塊）
- 한 잔
- 두 잔
- 하나

3. Option Chunks（選項詞塊）
- 아이스
- 핫
- 포장
- 매장

4. Payment Phrases（付款短語）
- 카드로 할게요
- 현금으로 할게요
- 얼마예요?

5. Response Chunks（高頻回應）
- 네
- 아니요
- 네, 맞아요
- 잠깐만요

### Suggested Review Fields (zh-TW)

- `review_summary_zh_tw`: 以點餐任務為中心整理功能詞塊，能直接支援拼句與店員問答回應。
- `novelty_rationale_zh_tw`: 不做純詞彙清單，改用「可輸出 chunk」分組，降低初級輸出負擔。

## 5. Output Nodes (Path Nodes)

## 5.1 Controlled Output A: Chunk Assembly

- `candidate_type`: `path_node`
- `learning_role`: `controlled_output`
- `output_mode`: `chunk_assembly`
- `skill_focus`: `speaking` (or `writing` in text mode)

### Task

用詞塊拼出「完整點餐句」。

### Example Item

給定詞塊：
- `아이스 아메리카노`
- `한 잔`
- `주세요`

目標：
- 組成一個自然點餐句（初級可接受模板化）

### Variants

- 加入第二品項（使用 `이랑/하고`）
- 改成外帶/內用情境提示

### Success Criteria

- 有品項 + 數量 + 請求
- 語序可接受（初級容錯）

## 5.2 Controlled Output B: Response Builder

- `candidate_type`: `path_node`
- `learning_role`: `controlled_output`
- `output_mode`: `response_builder`
- `skill_focus`: `speaking`

### Task

看到店員提問，拼出短回應。

### Example Prompts

1. 店員：`매장에서 드세요, 아니면 포장이세요?`
- 目標回應：`포장이요.` / `매장에서 먹을게요.`（初級版本可簡化）

2. 店員：`결제는 카드로 하세요?`
- 目標回應：`네, 카드로 할게요.`

### QA Focus

- 回應應短而高頻，不要要求初級一次產長句

## 5.3 Immersion Output A: Guided Speaking Role-play

- `candidate_type`: `path_node`
- `learning_role`: `immersion_output`
- `output_mode`: `guided`
- `skill_focus`: `speaking`

### Task Brief

你在咖啡廳點餐，請完成點餐流程。

### Prompt Card (3 Constraints)

- 預算：`10,000 韓元內`
- 飲料：`熱飲`
- 需求：`外帶`

### Required Patterns (must use at least 2)

- 點餐請求句
- 外帶/內用回應
- 付款方式回應

### Scoring Rubric (review-friendly)

- `fit`: 是否完成點餐任務
- `learnability`: 是否使用本單元句型
- `engagement`: 情境是否自然

## 5.4 Immersion Output B: Guided Writing (Message)

- `candidate_type`: `path_node`
- `learning_role`: `immersion_output`
- `output_mode`: `guided`
- `skill_focus`: `writing`

### Task

寫 1-2 句訊息給朋友，確認在咖啡廳見面時間與地點。

### Prompt

- 地點：`站前咖啡廳`
- 時間：`晚上 7 點`
- 目標：`確認對方是否方便`

### Acceptable Output Shape (beginner)

- 提議/確認一句
- 時間或地點一句

### Why It Belongs Here

- 同主題（咖啡廳），但不是重複點餐對話
- 放大書面輸出比例，且仍在初級負擔範圍內

## 5.5 Same-day Review Retrieval Node

- `candidate_type`: `path_node`
- `learning_role`: `review_retrieval`
- `output_mode`: `review_retrieval`
- `skill_focus`: `integrated`

### Goal

當日回想，不重看整段對話。

### Retrieval Prompts (minimal cues)

1. 你要點「冰美式一杯」怎麼說？（可用句框提示）
2. 店員問內用/外帶，你怎麼回「外帶」？
3. 怎麼說「刷卡可以嗎？」或「我刷卡」？
4. 用一句話跟朋友確認 7 點在咖啡廳見面

### QA Note

- 這是 retrieval，不是再次閱讀題
- 不應只要求選擇題辨識

## 5.6 Cross-unit Transfer Node (Scheduled, +1 / +3)

- `candidate_type`: `path_node`
- `learning_role`: `cross_unit_transfer`
- `output_mode`: `transform` or `guided`
- `skill_focus`: `speaking`
