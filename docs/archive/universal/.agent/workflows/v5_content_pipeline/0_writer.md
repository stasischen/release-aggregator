# Skill: Phase 0 Content Writer (內容寫作)

這是內容生產的第一步。本技能定義了如何撰寫符合 V5 管線標準的原始文檔 (Source Documents)。
This is the first step of content production. This skill defines how to write source documents that meet V5 pipeline standards.

## 🎯 Objective

撰寫正確格式的 `.yarn` (對話) 與 `.md` (文章) 檔案，確保後續管線能正確解析與處理。
Write correctly formatted `.yarn` (dialogue) and `.md` (article) files to ensure correct parsing and processing by the pipeline.
